# ConnectWallet
A simple Web3 button to connect your wallet that you can add to any Web3 app or website. 
Compatible wallets include:
- Metamask
- Coinbase Wallet
- WalletConnect
- Trust Wallet
- Rainbow
- Zerion Wallet
- Phantom
